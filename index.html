onclick, onchange, oninput, ondblclick

onclick

Az onclick azt csinálja, hogy amit rá nyomunk a gombra le futtatja a function-t. 
                        
<button onclick="OnClickButton()">onclick</button>
                        
                    
                        
function OnClickButton() {
    console.log("I like cows")
}

onchange

Az on change azt csinálja hogy amint a user abba hagyta az adatok változtatását akkor futtatja le a fucntion-t. 
                        
<input type="range" onchange="OnChangeInput()" id="ID">
                        
                    
                        
function OnChangeInput() {
    console.log(document.getElementById('ID').value)
}

oninput

Az oninput azt csinálja hogy ha van változás az inputtal ez esetben akkor aktiválódik a function. 
                        
<input type="range" onchange="OnInputInput()" id="ID">
                        
                    
                        
function OnInputInput() {
    console.log(document.getElementById('ID').value)
}

ondbclick

Az ondbclick azt csinálja ha duplán kattintunk egy gombra akkor le futtatja valamit. 
                        
<button ondblclick="OnDbClickButton()">onclick</button>
                        
                    
                        
function OnDbClickButton() {
    console.log("I like cows")
}
