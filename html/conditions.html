feltételek

if

Az if egy feltétel ami képes el dönteni és külömbséget tenni 2 érték között. 
                    
const num1 = 10
const num2 = 20

if(num1 > num2) {
console.log("num1 a nagyobb szám")
} else if(num1 < num2) {
console.log("num2 a nagyobb szám")
} else {
console.log("A két szám eggyenlő")
}

else if

Az else if hasonlóan az if hez képes eldönteni és külömbséget tenni 2 érték között, viszont az else if csak akkor jön szóba ha az elsődleges feltételnek nem felel meg az érték és több mint egy feltételt adunk meg. 
                    
const num1 = 10
const num2 = 20

if(num1 > num2) {
console.log("num1 a nagyobb szám")
} else if(num1 < num2) {
console.log("num2 a nagyobb szám")
} else {
console.log("A két szám eggyenlő")
}

else

Az else egy olyan feltétel ami akkor lép értékbe ha semelyik feltételnek nem felel meg az érték. 
                    
const num1 = 10
const num2 = 20

if(num1 > num2) {
console.log("num1 a nagyobb szám")
} else if(num1 < num2) {
console.log("num2 a nagyobb szám")
} else {
console.log("A két szám eggyenlő")
}
